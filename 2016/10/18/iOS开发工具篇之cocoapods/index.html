iOS开发工具篇之cocoapods

cocoapods使用教程

如果你是一名具有一年或者迭代过一个项目三个版本以上的ios开发者我想你对友盟,极光,shareSDK….一定不陌生,可是如果他们版本更新你该怎么办?
如果你说就这么放着不管,那么恭喜你你失去了一个程序员最基本的求知欲.如果你手动删除从新在添加,那我只能夸你手太勤快,但是脑袋太懒.如果你遇到sdk更新的问题只需要百度一下就会看到排山蹈海的cocoapods的使用教程.

说了那么多那么cocoapods到底是干嘛的?就是第三方库管理软件只需要简单的指令就可以实现第三方库的添加,移除,更新.到底有多方便谁用谁知道.

安装cocoapods的过程很简单但是也有很多坑只有自己走过才能体会其中的辛酸

第一步安装cocoapods

sudo gem install cocoapods

由于cocoapods使用的是国外的源在我们大天国是不允许使用的除非你翻墙,但是我们有更好的办法就是替换成淘宝源

第二步替换淘宝源

1.移除原有源

gem source --remove https://rubygems.org/

2.替换淘宝源

gem source -a https://ruby.taobao.org/

3.验证是否替换成功

gem source -l

4.最后就可以通过 sudo gem install cocoapods 正常安装 cocoapods 了

第三步如何使用cocoapods

要使用cocoapods,必须要有一个Podfile文件

cd "你自己的工程目录"

touch Podfile //创建一个Podfile文件

第四步搜索想要添加的库

pod search AFNetworking //搜索想要添加的库

vi Podfile //进入编辑模式

//添加库
pod 'AFNetworking', '~> 2.0'

然后按下 esc 键,输入 :wq 完成保存,在终端输入 pod install 完成安装

在安装完成之后我们的工程里会多出一个 xcworkspace 的文件,以后打开这个文件就可以了

第三方库的使用

我们已经将第三方库添加完了可是他的使用和手动添加的有什么不一样那?

#import <要引用的库名>

就这样以后就可以告别手动添加库的烦恼了
